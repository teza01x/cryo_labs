# Cryo Labs — Portfolio Website (React + JS)

Базовая статическая одностраничная верстка портфолио агентства "Cryo Labs".

Как запустить локально:

```powershell
cd "c:\Users\primarch\Desktop\JS projects\cryo_labs"
npm install
npm run dev
```

Файлы:
- `index.html` — точка входа
- `src/main.jsx` — инициализация React
- `src/App.jsx` — основной компонент
- `src/components/*` — UI-компоненты
- `src/styles.css` — базовые стили

Если нужно, могу добавить: анимации, развертывание (Vercel/Netlify), контактную форму или Web3-интеграции.
